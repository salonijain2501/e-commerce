# E-Commerce Web Application

A full-stack e-commerce web application currently under development.  
The project is focused on building a responsive and user-friendly **frontend**, with backend services, payment integration, and chatbot support planned for upcoming phases.

## Tech Stack
### Frontend
- React
- Tailwind CSS
- JavaScript

### Backend (Planned)
- Node.js
- Express.js
- MongoDB

### Payment Gateway (Planned)
- Razorpay

### Chatbot Integration (Planned)

## How to Run the Project Locally

Clone the repository:
```bash
git clone https://github.com/salonijain2501/e-commerce.git
```
Navigate to the project folder:
```bash
cd e-commerce
```
Install dependencies:
```bash
npm install
```
Start the development server:
```bash
npm start
```
